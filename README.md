# SauceDemo Automation Project

## 🚀 Description
This project contains automated UI tests for the SauceDemo website using WebdriverIO.

## 🛠️ Setup

1. Install dependencies:
   
    npm install

2. Run tests:

   wdio run ./wdio.conf.ts

3. Generate and open Allure report:

   npm run report:generate && npm run report:open

4. Alternatively, serve Allure report in one command:

   npm run allure
